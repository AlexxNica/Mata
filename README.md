## Mata
A Chrome extension that makes reading the web friendly for your eyes. 


## Installation
[Download it on the Chrome Store](https://chrome.google.com/webstore/detail/mata/oakhnpcgdembaopakdnfkejfcaeipeik?hl=en-US)
